# Web-System

Este repositório foi separado em duas aplicações:

- `frontend/`: aplicação Angular.
- `backend/`: API Node.js com NestJS.

## Estrutura

```text
web-system/
  frontend/
  backend/
```

## Executar o Frontend (Angular)

```bash
cd frontend
npm install
npm start
```

Frontend em: `http://localhost:4200`

## Executar o Backend (NestJS)

```bash
cd backend
npm install
npm run start:dev
```

Backend em: `http://localhost:3000`
